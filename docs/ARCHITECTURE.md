## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 编程语言 | Python 3.10+ | 主要开发语言 |
| 深度学习 | PyTorch 2.0+ | 深度学习框架 |
| 模型库 | Transformers 4.40+ | 预训练模型 |
| 微调框架 | PEFT, MS-Swift, LLaMA-Factory | 参数高效微调 |
| API框架 | FastAPI | REST API服务 |
| 推理加速 | vLLM | 高性能推理 |
| 容器化 | Docker, Docker Compose | 部署和隔离 |
| 监控 | Prometheus, Grafana | 指标收集和可视化 |
| 日志 | Python Logging | 日志管理 |
| 测试 | pytest | 单元测试 |
